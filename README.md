# Todo-List

Features:

Create and delete projects
Create, edit and delete tasks
Mark tasks complete
Automatic project completion when all tasks are complete
Priority tasks
Due dates with countdown
Task details
Local storage persistence
Responsive layout

Built With:

JavaScript (ES Modules)
Webpack
CSS Grid & Flexbox
date-fns
Local Storage API

What I Learned:

Organising an application into modules
Separating DOM manipulation from application logic
Managing application state
Using Webpack to bundle a project
Persisting data with Local Storage
Refactoring duplicated logic

Future Improvements:

Task categories or tags
Drag-and-drop task ordering
Task search/filtering
Dark/light theme
Keyboard shortcuts
